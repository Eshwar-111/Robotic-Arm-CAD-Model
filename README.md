# 🤖 6DOF Robotic Arm CAD Model

*A complete CAD design and assembly of a 6 Degrees of Freedom robotic arm built using SolidWorks.*



## 📖 About the Project

This repository contains the complete CAD model and assembly of a **6DOF Robotic Arm** designed in **SolidWorks**.

The goal of this project wasn't just to make something that looks cool (although... it does 😎), but to understand how industrial robotic arms are designed from the ground up.

Every part was modeled individually before being assembled into a complete robotic arm.

---

## 🎯 Objectives

- Learn professional CAD modeling
- Understand robotic arm mechanisms
- Practice SolidWorks Assembly
- Build a strong robotics design portfolio
- Prepare for Motion Study and FEA Analysis
